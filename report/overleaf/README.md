## Contents
- `main.tex` — main IEEE report source
- `references.bib` — bibliography entries
- `figures/` — copied plots and qualitative images used in the report
- `main.pdf` — locally compiled reference PDF

## Recommended upload artifact
If you want the fastest path into Overleaf, upload:

- `../phasephyto_overleaf_package.zip`

## How to use in Overleaf
1. Open the IEEE conference template you were given.
2. Create a new Overleaf project or upload this folder as a zip.
3. Replace the template `main.tex` with this `main.tex`.
4. Keep `references.bib` and the `figures/` folder in the project.
5. Compile with pdfLaTeX.
